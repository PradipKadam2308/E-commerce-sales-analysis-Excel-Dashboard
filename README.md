# # 📊 E-Commerce Sales Analysis Dashboard

## 🚀 Project Overview

This project presents an interactive **E-Commerce Sales Analysis Dashboard** developed in Microsoft Excel to analyze business performance across sales, profit, orders, products, categories, and regions.

The dashboard helps stakeholders monitor key business metrics, identify profitable categories, track sales trends, and make data-driven decisions.

---

- 📈 Total Profit: $17,691.02
